Unity Engine

TextMesh Pro

Unity Technologies ApS

Networking

Photon PUN 2

Exit Games GmbH
Provides real-time multiplayer networking for all co-op sessions, room management, and player synchronization.

License & Terms: Photon privacy & cookie policy

Unity Gaming Services

Unity Leaderboards (com.unity.services.leaderboards)

Unity Technologies ApS
Powers the global leaderboard for Survival mode. Stores anonymous player IDs and scores.

License & Privacy: unity.com/legal/privacy-policy

Unity Authentication (com.unity.services.authentication)

Unity Technologies ApS
Provides anonymous device-based authentication used to associate leaderboard entries with a player's device. No account, email, or personal data is required.

License & Privacy: unity.com/legal/privacy-policy

Unity Services Core (com.unity.services.core)

Unity Technologies ApS
Shared initialization runtime required by Unity Gaming Services packages.

License: unity.com/legal

Unity Analytics

Unity Technologies ApS
Collects gameplay events and session data to help improve game balance and performance — only after you opt in. What it processes is detailed in our Privacy Policy.

License & Privacy: unity.com/legal/privacy-policy
